About InstaBooth ✨

InstaBooth is a digital photobooth experience designed to help users capture, customize and download personalized photostrips directly from their browser. With filters, backgrounds, stickers and creative editing tools, users can transform simple photos into memorable keepsakes.

About the Developer 👩‍💻

Hi! I'm Devika, a BCA student with a growing interest in technology, programming and creative problem-solving. I enjoy exploring databases, networking, Python and various areas of computer science while building creative projects and learning new technologies.

Project Journey 🚀

InstaBooth was created to combine creativity and technology through an interactive photobooth experience. This project helped me learn more about JavaScript, browser APIs, image manipulation, user interaction and responsive web design while creating something fun and useful.
